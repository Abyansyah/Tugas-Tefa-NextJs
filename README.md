# Website GJI.

Website online store yang khsusus menjual drone

## Run Locally

Clone the project

```bash
  git clone https://github.com/Abyansyah/Tugas-Tefa-NextJs.git
```

Go to the project directory

```bash
  cd Tugas-Tefa-NextJs
```

Install dependencies

```bash
  npx create-next-app .
```

Start the server

```bash
  npm run dev
```
